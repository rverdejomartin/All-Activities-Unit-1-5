Web buscadora de películas

Funcionamiento principal de la web

  • Esta web implementa la obtención de datos mediante una petición.
  • Se obtendran dichos datos de una API.
  • Dispone de 4 elementos, el input de busqueda de nombre, el de año, para buscar, y un desplegable para señalar el tipo de busqueda, pelicula o serie.
  • Se podrá hacer click sobre la imagen de cualquier película para poder ver información sobre cierta película.
